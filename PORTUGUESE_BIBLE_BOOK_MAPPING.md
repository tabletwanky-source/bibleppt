## Special Cases

### Books with Accents

**Critical:** Portuguese accents must be preserved in the mapping:

- ✅ `Gênesis` (correct)
- ❌ `Genesis` (will match English instead)
- ✅ `Êxodo` (correct)
- ❌ `Exodo` (won't match)

The system handles these intelligently:
- User can type with or without accents
- Normalization function matches both
- API always receives English names

### Books with Same Spelling

Some Portuguese book names are identical to English:

- Daniel = Daniel
- Joel = Joel

The system handles these correctly by checking Portuguese first, then falling back to English.

### Numbered Books

Format: `{Number} {BookName}`

Examples:
- `1 João 4:19` → `1 John 4:19`
- `2 Reis 2:11` → `2 Kings 2:11`
- `1 Tessalonicenses 5:16` → `1 Thessalonians 5:16`

## Implementation Details

### Code Location

**File:** `src/services/bibleNormalization.ts`

```typescript
export const BIBLE_BOOKS_MAPPING: Record<string, string[]> = {
  // Format: "English": ["French", "Spanish", "Portuguese"]
  "Genesis": ["Genèse", "Génesis", "Gênesis"],
  "Exodus": ["Exode", "Éxodo", "Êxodo"],
  // ... all 66 books
};
```

### Normalization Function

```typescript
export function normalizeBibleReference(input: string): string {
  // 1. Trim and lowercase input
  const trimmedInput = input.trim();
  const lowerInput = trimmedInput.toLowerCase();

  // 2. Check each English book and its translations
  for (const [english, translations] of Object.entries(BIBLE_BOOKS_MAPPING)) {
    // 3. Check if input starts with Portuguese translation
    for (const translation of translations) {
      if (lowerInput.startsWith(translation.toLowerCase())) {
        // 4. Replace Portuguese with English
        const regex = new RegExp(`^${translation}`, "i");
        const normalized = trimmedInput.replace(regex, english);
        return normalized;
      }
    }
  }

  // 5. Return original if no match found
  return trimmedInput;
}
```

### Key Features

1. **Case Insensitive:** "joão" = "João" = "JOÃO"
2. **Accent Aware:** "Genesis" matches "Gênesis"
3. **Prefix Matching:** Only matches at start of string
4. **Preserves References:** "João 3:16" → "John 3:16" (keeps 3:16)
