# 📱 Guide Rapid: BibleSlide Remote PRO

## ⚡ Quick Start (5 Minit)

### Pou Prezantè:

```
1. Click Present → Sesyon kreye
2. Wè bouton vèt ak kod (ex: AB4X9Q)
3. Badge montre konbyen aparèy konekte
4. Click bouton Lock/Unlock pou kontrole
```

### Pou Telefòn:

```
1. Ale /remote
2. Antre kod 6 lèt
3. Press Next/Previous
4. Wè device count nan header
```
