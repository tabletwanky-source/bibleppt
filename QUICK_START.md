# BibleSlide - Quick Start Guide

## 🚀 Get Started in 5 Minutes

### 1. Access the Application

Visit **https://bibleslide.org** or run locally:

```bash
npm install
npm run dev
```

### 2. Create an Account

1. Click "Login" in the top right
2. Choose "Sign Up"
3. Enter email and password
4. Verify your email (if required)

### 3. Create Your First Presentation

#### Option A: Bible Verses

1. Go to **Dashboard** → **Presentations**
2. Click **"New Presentation"**
3. Click **"Search Bible"** (from existing search)
4. Enter: `John 3:16-17`
5. Click **"Generate Slides"**
6. Each verse becomes one slide automatically!

#### Option B: Song Lyrics

1. Go to **Dashboard** → **Presentations**
2. Click **"New Presentation"**
3. Click **"Lyrics"** button
4. Paste your song lyrics:
   ```
   Amazing Grace

   Amazing grace how sweet the sound
   That saved a wretch like me

   I once was lost but now I'm found
   Was blind but now I see
   ```
5. Enable **"Auto-split"**
6. Click **"Generate Slides"**
7. Done! Each paragraph becomes a slide

### 4. Customize Your Slides

1. Click **"Themes"** button
2. Choose from:
   - **Classic** - Traditional black & white
   - **Modern** - Contemporary gradient
   - **Minimal** - Clean light design
   - **Church Dark** - Elegant serif style
3. Click **"Media"** to add backgrounds:
   - Upload images
   - Upload videos
   - Choose gradient presets
4. Click **"Save"** to save your work

### 5. Present Live

1. Click **"Present"** button
2. Full-screen mode activates
3. Use keyboard to navigate:
   - **Right Arrow** or **Space** → Next slide
   - **Left Arrow** → Previous slide
   - **F** → Toggle fullscreen
   - **Esc** → Exit

### 6. Control from Your Phone

**Presenter's Device:**
1. In presentation mode, click **📱 icon**
2. A QR code appears with session code (e.g., **AB23CD**)

**Controller's Phone:**
1. Scan QR code OR visit `bibleslide.org/remote?session=AB23CD`
2. Wait for "Connected" status
3. Use **Next** and **Previous** buttons
4. Slides change instantly on main screen!

### 7. Share with Online Viewers

1. Share viewer link: `bibleslide.org/viewer?session=AB23CD`
2. Viewers see slides in real-time
3. Perfect for:
   - Online services
   - Musicians following along
   - Pastors with notes
   - Remote team members

## 📱 Mobile Quick Start

### Create Presentation (Mobile)
1. Login on your phone
2. Tap **Presentations**
3. Tap **+ New**
4. Add lyrics or verses
5. Choose theme
6. Tap **Save**

### Present (Tablet/Phone)
1. Open presentation
2. Tap **Present**
3. Swipe or use buttons to navigate
4. Full-screen supported

### Remote Control (Phone)
1. Scan presenter's QR code
2. Tap **Next** / **Previous**
3. That's it!

## 🎓 Common Workflows

### Sunday Sermon

```
1. Create presentation: "Sunday Sermon - John 3"
2. Add Bible verses: John 3:1-16 (auto-generates 16 slides)
3. Add title slide with sermon topic
4. Choose "Church Dark" theme
5. Upload church logo as background
6. Save presentation
7. On Sunday: Click "Present"
8. Control from phone while preaching
```

### Worship Songs

```
1. Create presentation: "Worship Set - Sunday"
2. Add Song 1 lyrics (auto-splits by verse/chorus)
3. Add Song 2 lyrics
4. Add Song 3 lyrics
5. Choose "Modern" theme
6. Add worship background image
7. Save
8. Present with keyboard or remote
```

### Bible Study

```
1. Create: "Bible Study - Romans 8"
2. Add verses: Romans 8:1-11
3. Add custom slides with discussion questions
4. Choose "Minimal" theme for readability
5. Share viewer link with participants
6. Everyone follows along on their devices
```

## ⚡ Power User Tips

### Keyboard Shortcuts (Presentation Mode)
- `Right Arrow` or `Space` → Next slide
- `Left Arrow` → Previous slide
- `F` → Fullscreen toggle
- `Esc` → Exit presentation

### Quick Actions
- **Duplicate** a presentation → Edit and reuse
- **Drag slides** to reorder in editor
- **Upload gradients** for modern look
- **Add church name** to every slide (in settings)

### Remote Control Tips
- Multiple phones can control simultaneously
- Session expires after 24 hours
- Local network mode works offline
- QR code updates if you restart presentation

### Viewer Mode Best Practices
- Share link before service starts
- Works on all devices (phone, tablet, laptop)
- No account needed for viewers
- Auto-reconnects if connection drops

## 🔧 Troubleshooting

### Slides Not Appearing?
- Make sure you clicked "Generate Slides"
- Check that you're logged in
- Try refreshing the page

### Remote Control Not Working?
- Verify session code is correct
- Check internet connection
- Make sure session is active (green dot)
- Try generating new session code

### Can't Upload Media?
- Check file size (must be < 50MB)
- Verify file type (JPG, PNG, MP4 only)
- Ensure you're logged in
- Check internet connection

### Presentation Won't Fullscreen?
- Press `F` key
- Or click fullscreen button
- Allow browser fullscreen permission
- Try pressing F11 (browser fullscreen)
